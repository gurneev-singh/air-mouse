# 🖱️ Air Mouse

Control your PC cursor, click and scroll using just hand gestures — no mouse needed!

## ✨ Features
- ☝️ **Index finger up** — move cursor
- 🤏 **Pinch** — left click
- ✌️ **Peace sign** — right click
- ✊ **Fist** — scroll up/down

## 🛠️ Tech Stack
- Python 3.11
- OpenCV
- MediaPipe
- PyAutoGUI

## 🚀 How to Run
```bash
pip install opencv-python mediapipe pyautogui numpy
python main.py
```

## 📁 Project Structure
```
air-mouse/
├── main.py              # Entry point
├── hand_tracker.py      # Hand detection and gesture recognition
├── mouse_controller.py  # Cursor movement and clicks
└── utils.py             # Helper functions
```
