# 🖱️ Air Mouse

Control your PC cursor, click and scroll using just hand gestures — no mouse needed!

## ✨ Features
- ☝️ **Index finger up** — move cursor
- 🤏 **Pinch** — left click
- ✌️ **Peace sign** — right click
- ✊ **Fist** — scroll up/down

## 🛠️ Tech Stack
- Python 3.11
- OpenCV
- MediaPipe
- PyAutoGUI
